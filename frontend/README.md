# Knoggin Frontend

The user interface for Knoggin, built with React, Vite, and Tailwind CSS.

> [!IMPORTANT]
> The **Knoggin Server** must be running for the frontend to fetch data. See [knoggin-server](../knoggin-server) for setup and start instructions.

## Features
- Interactive knowledge graph visualization.
- Entity management and merge proposal review.
- Onboarding flow for LLM configuration.
- Real-time agent chat interface.

## Setup

1.  **Install dependencies:**
    ```bash
    npm install
    ```

2.  **Start development server:**
    ```bash
    npm run dev
    ```

3.  **Build for production:**
    ```bash
    npm run build
    ```

## Stack
- **Framework:** React
- **Build Tool:** Vite
- **Styling:** Tailwind CSS, shadcn/ui
- **Icons:** Lucide React
